# RSS reader

RSS reader - frontend project for fetching and displaying RSS-feeds in user-friendly way

## Requirements:

* Node.js >= 14
* npm >= 6
* make >= 4

## Installation and startup

```shell
make install

make build
```

## Eslint check

```shell
make lint
```
